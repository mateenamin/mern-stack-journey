Node.js ne JavaScript ko server pe chalana possible kiya! Simple Example: Pehle: JavaScript → Sirf Browser ✅ Node.js ke baad: JavaScript → Browser ✅ JavaScript → Server ✅ JavaScript → Database ✅ Install Check Karo: node --version # v18+ hona chahiye npm --version # 9+ hona chahiye Pehla Node.js File: server.js banao: // Browser mein nahi — Terminal mein chalega! console.log("Hello Node.js!") console.log("Main server pe chal raha hoon!") Terminal mein chalao: node server.js Node.js vs Browser JS: Browser JS Node.js document ✅ ❌ window ✅ ❌ fs (file system) ❌ ✅ http (server) ❌ ✅ console.log ✅ ✅ require vs import: // Node.js purana tarika — CommonJS const fs = require("fs") const express = require("express") // Naya tarika — ES Modules import fs from "fs" import express from "express" Hum ES Modules use karenge — React jaisa! Built-in Modules: 1 — fs — File padhna likhna: const fs = require("fs") // File likhna fs.writeFileSync("test.txt", "Hello Mateen!") // File padhna const data = fs.readFileSync("test.txt", "utf8") console.log(data) // Hello Mateen! 2 — path — File path: const path = require("path") console.log(path.join("folder", "file.txt")) // folder/file.txt 3 — http — Simple Server: const http = require("http") const server = http.createServer((req, res) => { res.write("Hello from Server!") res.end() }) server.listen(3000, () => { console.log("Server chal raha hai: http://localhost:3000") }) Browser mein localhost:3000 kholo — Hello from Server! dikhega! Honest Baat: ✅ fs, path, http — sirf concept ke liye hain ✅ Real projects mein Express use karte hain ✅ Yeh directly use nahi hota! Sirf Yaad Rakho: Node.js = JavaScript ko server pe chalata hai Bas itna enough hai!
