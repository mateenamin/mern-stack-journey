Topic 01 Setup + Basic Routes
React Router = Bina page reload ke ek page se doosre pe jaana!
Install
npm install react-router-dom
App.jsx
import { BrowserRouter, Routes, Route } from "react-router-dom"

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/"        element={<Home />} />
        <Route path="/about"    element={<About />} />
        <Route path="/projects" element={<Projects />} />
      </Routes>
    </BrowserRouter>
  )
}

Topic 02 Link + NavLink
Link = Page reload nahi — seedha navigate!
NavLink = Active page pe style change!
Farq
// ❌ Normal a tag — page reload hoga!
<a href="/about">About</a>

// ✅ Link — page reload nahi!
<Link to="/about">About</Link>

// ✅ NavLink — active style ke saath!
<NavLink to="/about"
  style={({ isActive }) => ({
    color: isActive ? "#c9a84c" : "white"
  })}>
  About
</NavLink>
Link
Simple navigate
NavLink
Active style milti hai
isActive
Current page = true
Topic 03 Dynamic Routes — :id
Dynamic Route = URL mein variable data pass karna!
/user/1, /user/2, /product/phone
Route Setup
// :id = dynamic part
<Route path="/user/:id" element={<UserDetail />} />
<Route path="/product/:name" element={<ProductDetail />} />
Link banao
// Template literal se dynamic link
<Link to={`/user/${user.id}`}>Detail</Link>
Topic 04 useNavigate
useNavigate = Code se navigate karo — Link ke bina!
Form submit, login, button click pe redirect!
Syntax
import { useNavigate } from "react-router-dom"

const navigate = useNavigate()

navigate("/")          // Home
navigate("/dashboard")  // Dashboard
navigate(-1)           // Wapas jao
navigate(1)            // Aage jao
✅ Login hone ke baad — Dashboard pe bhejo → navigate("/dashboard")
Topic 05 useParams
useParams = URL se data nikalo!
URL: /user/5 → id = "5"
Syntax
import { useParams } from "react-router-dom"

function UserDetail() {
  const { id } = useParams()
  // URL: /user/5 → id = "5"
  // URL: /user/ali → id = "ali"

  const user = users.find(u => u.id === parseInt(id))
}
⚠️ useParams string return karta hai — number chahiye toh parseInt() karo!
Topic 06 Nested Routes + Outlet
Nested Routes = Route ke andar Route!
/dashboard/profile, /dashboard/settings
App.jsx
<Route path="/dashboard" element={<Dashboard />}>
  <Route path="profile"  element={<Profile />}  />
  <Route path="settings" element={<Settings />} />
</Route>
Dashboard.jsx — Outlet zaroori hai!
import { Outlet } from "react-router-dom"

function Dashboard() {
  return (
    <div>
      <h1>Dashboard</h1>
      <Outlet /> // ← Child yahan dikhega!
    </div>
  )
}
✅ Outlet = Child page yahan dikhao! Bina Outlet ke nested pages nahi dikhengi!
Topic 07 Protected Routes
Protected Route = Login ke bina page nahi kholne!
Login nahi kiya → Login page pe bhejo!
ProtectedRoute.jsx
import { Navigate } from "react-router-dom"

function ProtectedRoute({ children }) {
  const isLoggedIn = localStorage.getItem("user")

  if (!isLoggedIn) {
    return <Navigate to="/login" />
  }

  return children
}
App.jsx mein use karo
<Route path="/dashboard"
  element={
    <ProtectedRoute>
      <Dashboard />
    </ProtectedRoute>
  }>
  ...
</Route>
Login karo → localStorage mein save
Dashboard kholne ki koshish
ProtectedRoute check karta hai
User hai ✅ → Dashboard dikhao
User nahi ❌ → Login pe bhejo
Topic 08 404 Page — Not Found
404 Page = Galat URL pe jaو — error page dikhao!
App.jsx — Hamesha Last Mein!
<Routes>
  <Route path="/"       element={<Home />}     />
  <Route path="/about"  element={<About />}    />
  // ⚠️ Hamesha LAST mein rakho!
  <Route path="*"       element={<NotFound />} />
</Routes>
⚠️ path="*" hamesha LAST mein rakho — warna sab routes block ho jayenge!
